# homebrew-drift

Homebrew tap for [drift](https://github.com/phlx0/drift) — a terminal screensaver that turns idle time into ambient art.

## Install

```bash
brew install phlx0/drift/drift
```

Or tap first, then install:

```bash
brew tap phlx0/drift
brew install drift
```

## Upgrade

```bash
brew upgrade drift
```

## Formulas

| Formula | Description |
|---------|-------------|
| `drift` | Terminal screensaver and ambient visualiser |
